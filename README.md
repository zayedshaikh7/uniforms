# I & S Global Enterprises Website

A production-ready, multi-page B2B catalog website for I & S Global Enterprises, a Mumbai-based uniform and workwear manufacturer established in 2001.

## Technology Stack

- **Frontend**: React 19
- **Build Tool**: Vite
- **Routing**: React Router 7
- **Styling**: Vanilla CSS

## Pages & Structure

- **Home** (`src/pages/Home.jsx`) - Modern landing page with scroll animations and dynamic stats.
- **About Us** (`src/pages/About.jsx`) - Company history and infrastructure details.
- **Products** (`src/pages/Products.jsx`) - Catalog of all uniform categories.
- **Category Pages** - Dedicated views for Industrial, Medical, Hospitality, and Corporate wear.
- **Contact Us** (`src/pages/Contact.jsx`) - Interactive contact page with **WhatsApp integration** for direct inquiries and Google Maps location.
